# ECHO

实时反馈墙项目。支持多人协作、实时同步和 AI 辅助整理。

ECHO is a realtime feedback wall for collaborative posting, live sync, and AI-assisted organization.

## Tech Stack / 技术栈

- Frontend: Vue 3, TypeScript, Vite
- Backend: FastAPI, SQLite, WebSocket
- AI: DeepSeek API
